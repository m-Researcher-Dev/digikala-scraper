
# 🖥️ Digikala Laptop Scraper / استخراج لپ‌تاپ از دیجی‌کالا

![Application Preview](screenshot.png)

---

# 🌍 English Documentation

## ✨ Features

- **Smart Browser Management**  
  Automatic Edge browser updates and dynamic WebDriver installation

- **Anti-Detection System**  
  Headless mode support and randomized user fingerprint simulation

- **Data Processing**  
  Scrape 1–40 pages of product listings and export to Excel with one-click download

---

## 🚀 Quick Start

### Install dependencies
```bash
pip install selenium streamlit openpyxl pandas psutil requests
```

Run application

```bash
streamlit run project_1.py
```

---

🧠 Core Components

· project_1.py – Main application logic
· selenium_setting.py – Browser automation setup
· kill_browser.py – Process and browser management
